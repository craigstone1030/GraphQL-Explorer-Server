# GraphQL-Explorer

This is a demo which contains GraphQL and framework based on it(or work with it perfectly), including these for now:

- TypeGraphQL
- Apollo-Client & React & Parcel
- Apollo-Server
- TypeORM + SQLite3
- TypeStack Product

## Progress: 50%

see [docs](docs/README.md) for API will be used in this demo.

## Explore

```bash
npm install
npm run dev
# or use 2 terminals
npm run dev:client
npm run dev:server
```
